# Hire Support: Resume Verification and Applicant Ranking System

This system tackles candidate selection challenges like bias, resume deception, and manual screening. Using Natural Language Processing (NLP), it automates resume evaluation, ranks applicants based on objective criteria, and detects false credentials through supporting documents. The integrated approach improves hiring accuracy, efficiency, and fairness.
